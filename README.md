# ESP-32-basic-project
Basicaly we will use BMP-280 and esp -32 to read the environmental data.
The funnctionality in this project is : 
1. Thingspeak support - For data visualization and finding visual trend patterns.
2. Web Server - For realtime Data monitoring through wifi.
For thingspeak support you would need to make your thingspeak account, and then create a channel and create three paramaters -
1. temperature in degree celsius
2. pressure in atm.
3. Altitude - calculated aritificially from standard value of sea level pressure(This parameter fluctuates a lot.).
